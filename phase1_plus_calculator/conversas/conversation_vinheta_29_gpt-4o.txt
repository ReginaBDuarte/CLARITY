=== SYSTEM ===
Função:  És um assistente virtual especializado em cardiologia preventiva que apoia a consulta médica.

Formato geral
- Usa Markdown; não incluas texto fora das secções acima
- Quando um valor não se encontra nos dados, escreve “desconhecido”
- Usa sempre “,” como separador decimal e “.” para milhares (padrão português)

Objetivo: Analisar a vinheta clínica abaixo, extrair de forma estruturada todos os fatores relevantes para avaliar o risco cardiovascular.

=== USER ===
Mulher de 63 anos, técnica de análises. Status pós-angioplastia electiva (circunflexa proximal) em contexto de dor torácica atípica há 2 anos. Neste momento assintomática. Tem feito caminhadas.

Antecedentes pessoais: ex-tabagismo (10 UMAs, parou há 2 anos), ansiedade. 

Medicação habitual: ácido acetilsalicílico 100mg, atorvastatina +ezetimibe 20+10mg, sertralina 25mg, alprazolam 0.25mg.

Ao exame objectivo: peso 53Kg (IMC 20.1); pressão arterial 114-66mmHg, frequência cardíaca 63bpm. Auscultação cardíaca sem alterações.

Análises: Hb 13.1g/dL, creatinina 0.63mg/dL, glicemia em jejum 80mg/dL, HbA1c 5.6%, triglicerideos 82mg/dL, colesterol total 149mg/dL, colesterol HDL 53mg/dL, colesterol LDL 80mg/dL, NT proBNP <75pg/mL.

Exames complementares de diagnóstico: electrocardiograma e ecocardiograma de esforço sem alterações significativas

=== USER ===
1. Extração estruturada dos fatores de risco:
- Preenche uma tabela em Markdown com os fatores de risco necessários ao cálculo do risco cardiovascular
- Para cada fator: Valor (tal como aparece na vinheta ou indica “desconhecido”); Unidade se aplicável (mmHg, mg/dL, anos); 
- Mantém exatamente o mesmo nome de cada variável indicado na lista-referência abaixo.
- Lista-referência (não alterar nomes):
[Fator de Risco -> Valores; Idade -> Numérico; Género->Masculino; Feminino; Tabagismo -> Não fumador; Ex-Fumador; Fumador; Pressão Arterial Sistólica -> Numérico; Colesterol Total -> Numérico, Colesterol HDL -> Numérico, Colesterol não HDL (se necessário calcular) -> Numérico.]


2. Verificação de apicabilidade do SCORE2 (2021 ESC Guidelines on cardiovascular disease prevention in clinical practice)

- Verifica se o SCORE2 é aplicável (Sim / Não).
- Se o SCORE2 não for aplicável, indica razão segundo as recomendações ESC 2021.
- Apresenta as saídas numa tabela Markdown: | SCORE2 Aplicável | Exceção (se não  aplicável)


3. Criação de um json file
- No final, cria um json file que sumarize as informações, com a seguinte estrutura:

- Indica o grau de confiança da decisão sobre categoria de risco indicada no ponto anterior (Escolhe apenas uma categoria -> Baixa, Intermédia, Alta, Muito-Alta).
- Apresenta as saídas numa tabela Markdown:  Categoria Risco | Grau de Confiança 


2. Breve explicação clínica

Máximo 150 palavras. Realça os fatores que mais contribuíram para a classificação de risco (p. ex. HTA mal controlada, tabagismo, LDL elevado, modificadores de risco, etc.).
